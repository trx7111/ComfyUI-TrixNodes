# ComfyUI-TrixNodes

[![English](https://img.shields.io/badge/Language-English-blue?style=for-the-badge)](README.md) [![Русский](https://img.shields.io/badge/Язык-Русский-red?style=for-the-badge)](README_RU.md)

Элегантный, премиальный и высокопроизводительный набор нод для управления рабочими процессами в [ComfyUI](https://github.com/comfyanonymous/ComfyUI). Группируйте ноды, удаленно отключайте/включайте (bypass/mute) элементы, мгновенно перемещайтесь по холсту и избавьтесь от запутанных проводов с помощью плавных, GPU-оптимизированных анимаций.

## 🌟 Ключевые возможности

- **Удобный пульт управления**: Удаленно переключайте состояния bypass/mute для нод и групп.
- **Быстрый фокус камеры**: Нажмите кнопку глаза `👁` рядом с любой целью, чтобы мгновенно переместить камеру на неё.
- **Скрытие проводов (WLinks)**: Скрывайте соединительные линии в один клик. Провода красиво проявляются в виде светящихся частиц только при наведении или клике.
- **Переключатель производительности**: Выбирайте между экономичным режимом (15 FPS) и максимальной плавностью на частоте вашего монитора (60+ FPS).

## 🔌 Визуальное скрытие связей (WLinks)

WLinks позволяет точечно очистить холст от лишних проводов.

### 1. Выборочное скрытие связей
Кликните правой кнопкой мыши по любой ноде холста и выберите `🌊 Trix Hide Links`, чтобы скрыть её провода.

![Right Click Hide](assets/3.jpg)

### 2. Временный показ при наведении
Наведение или клик по ноде временно показывает скрытые связи.

![Hover Reveal](assets/4.jpg)

### 3. Светящиеся анимации связей
Настраивайте визуальные эффекты и траектории светящихся частиц.

![Glow Wires](assets/5.jpg)

### 4. Настройка индикаторов слотов
Выбирайте формы (WiFi Icon, Circle, Dashed Circle, Triangle) и пресеты цветов.

![Slot Indicators](assets/6.jpg)

## ⚙️ Глобальные настройки

Вы можете настроить параметры в стандартных настройках ComfyUI во вкладке **`Trix Nodes`**:

![Settings Panel 1](assets/7.jpg)

![Settings Panel 2](assets/8.jpg)

- **Slot Indicator Shape**: Форма индикатора слота (`WiFi Icon`, `Circle`, `Dashed Circle`, `Triangle`).
- **Color Mode**: Цвета линий (`Match Slot Color` под цвет типа слота или неоновые пресеты).
- **Animation Style**: Стиль эффекта (`Pulsation (Pulse)`, `Color Flow`, `Jelly/Water Wave Warping`, `Neon Plasma Flow`, `Sparkling Electricity`, `Floating Particles` или `Static`).
- **Wire Display Mode**: Когда показывать скрытые провода (`Hide Always`, `Show on Click` или `Show on Hover`).
- **Smooth animations (GPU-heavy)**: Переключение между экономией ресурсов (15 FPS) и максимальной плавностью на частоте монитора (60+ FPS).
- **Опции контекстного меню**: Управление показом пунктов при правом клике (`Show/Hide Bypasser controls`, `Copy Node ID`, `Copy Selected Node IDs` и `Show/Hide links`).

## ⚖️ Преимущества и сравнение

- **Чистота рабочего пространства**: Заменяет громоздкие стандартные провода ComfyUI на интерактивные, проявляющиеся при необходимости связи.
- **Удобная навигация**: Перемещение камеры по кнопке с глазом избавляет от необходимости вручную скроллить холст в поисках нужной ноды.
- **Гибкость управления**: Управляйте нодами и группами из одного пульта, расположенного в любой части схемы.

## 🛠️ Установка

1. Откройте терминал в папке кастомных нод ComfyUI:
   ```bash
   cd ComfyUI/custom_nodes/
   ```
2. Склонируйте этот репозиторий:
   ```bash
   git clone https://github.com/pixaroma/ComfyUI-TrixNodes.git
   ```
3. Перезапустите ComfyUI и обновите вкладку в браузере.
